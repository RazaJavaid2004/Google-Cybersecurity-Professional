# 📘 Cybersecurity Learning Portfolio

This folder contains **PowerPoint presentations** and **Word documents** created during the Google Cybersecurity Professional Certificate.  
Each file highlights applied skills in audits, incident response, threat modeling, and network analysis.

---

## 📂 File Index

### PowerPoint Presentations (`.pptx`)

#### 1. Security Audit Presentation
- **Purpose:** Summarizes findings from a compliance-aligned security audit.  
- **Highlights:**
  - Risk register with likelihood/impact scoring
  - Audit methodology (NIST/ISO framework alignment)
  - Recommendations for mitigation

#### 2. Incident Response Playbook
- **Purpose:** Visual walkthrough of the NIST Incident Response Lifecycle.  
- **Highlights:**
  - Preparation steps (policies, tools, training)
  - Detection & containment strategies
  - Recovery and lessons learned

#### 3. Threat Modeling Diagrams
- **Purpose:** STRIDE-based threat modeling for a sample web application.  
- **Highlights:**
  - Attack vectors (XSS, SQL injection, privilege escalation)
  - Mitigation strategies
  - Diagrammatic representation of system components

---

### Word Documents (`.docx`)

#### 1. Risk Register Report
- **Purpose:** Detailed documentation of risks identified during audit.  
- **Highlights:**
  - Risk ID, category, likelihood, impact
  - Assigned owner and mitigation plan
  - Compliance references (ISO 27001, CIS benchmarks)

#### 2. Incident Response Journal
- **Purpose:** Case study documenting a simulated security incident.  
- **Highlights:**
  - Timeline of events
  - Actions taken at each stage
  - Lessons learned and future improvements

#### 3. Network Traffic Analysis Report
- **Purpose:** Comparative analysis using Wireshark and tcpdump.  
- **Highlights:**
  - Packet capture samples
  - Anomaly detection in HTTP/DNS traffic
  - Recommendations for monitoring

#### 4. SQL Log Analysis Report
- **Purpose:** Queries and findings from log analysis.  
- **Highlights:**
  - Failed login detection
  - Privilege misuse identification
  - Automated anomaly detection queries

---

## 🚀 How to Navigate

- Open `.pptx` files for **visual presentations** of audits, playbooks, and threat models.  
- Open `.docx` files for **detailed written reports** with evidence, analysis, and recommendations.  
- Each file is modularized to demonstrate a specific cybersecurity skill.
